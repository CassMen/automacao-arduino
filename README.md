# Automação - Arduino
Projetos de automação residencial com Arduino.


## Escadas Iluminadas
### [Vídeo](https://www.youtube.com/watch?v=p79MFRNfi_8)

### Componentes:
- Arduino UNO
- Fonte chaveada 12V 5A
- Transistor Mosfet canal N IRF 530 
- Sensor PIR HCSR501
- Fita LED 3528 branco frio
- Cabos diversos
